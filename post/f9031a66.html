-m 参数

python xxx.py 与 python -m xxx.py ，这两种运行 Python 程序的方式的不同点在于，一种是直接运行，一种是当做模块来运行。

  • 直接运行的时候，xxx.py 所在目录是 sys.path
  • 以模块运行的时候，当前工作路径是 sys.path
